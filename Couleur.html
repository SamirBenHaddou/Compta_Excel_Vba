Couleur

Cette fonction permet de trouver la couleur d'une cellule : 

        Function couleur(Cellule As Range)
        Application.Volatile
        couleur = Cellule.Interior.ColorIndex
    End Function
          ‘Sert à compter toutes les valeurs verte par exemple
